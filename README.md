# ASV Aerocat V2

The **ASV Aerocat V2** is a small-scale autonomous surface vessel with a
catamaran hull and aerial propulsion. The project was developed for autonomous
navigation and for applications involving cooperation with aerial vehicles,
providing an accessible, low-cost platform for academic, educational, and
scientific research purposes.

## Repository Contents

This repository contains the main files required to build and replicate the
ASV Aerocat V2, including:

- assembly step-by-step instructions;
- STL files for the mechanical parts;
- PCB machining/fabrication files;
- project reference images.
