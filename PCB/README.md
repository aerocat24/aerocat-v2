# PCB and Electronics

This folder contains the PCB files and documentation for the electronic system
used in the ASV AeroCat V2. The board was developed to integrate the ESP32,
navigation sensors, propulsion interfaces, and communication resources into a
single compact platform.

Reference images of the PCB and assembled electronics are available in the
[`images/`](images/) folder.

## Overview

The electronic system is responsible for:

- Reading navigation and orientation sensors
- Controlling the propulsion system
- Managing wireless communication
- Distributing power to the onboard devices

The PCB replaces external jumper-based connections used in previous versions,
resulting in a cleaner and more organized integration between the components.

## Propulsion system

The vessel uses a differential aerial propulsion configuration with two
brushless thrusters. Each propulsion unit is composed of one brushless motor,
one ESC, and one propeller.

The independent control of the left and right motors allows the vehicle to
produce forward motion and turning maneuvers.

### Components

| Component | Model | Quantity |
| --- | --- | ---: |
| ESC | Mistery 30A | 2 |
| Brushless motor | EMax CF2822 – 1200 kV | 2 |
| Propeller | 8x4.5” | 2 |

### Propulsion data

| Parameter | Value |
| --- | --- |
| Distance between propellers | 45 cm |
| Maximum surge force | 9.43 N |
| Maximum yaw torque | ±1.06 N·m |

## Navigation system

The navigation system is based on the ESP32 WROOM 32 platform and includes GPS,
compass, and inertial sensing devices connected directly to the PCB.

### Navigation components

| Component | Function |
| --- | --- |
| ESP32 WROOM 32 | Main controller |
| GY-NEO6MV2 GPS | Position estimation |
| HMC5883L | Heading measurement |
| MPU-6050 | Inertial sensing |
| PC817 optocouplers | Electrical isolation between ESP32 and ESCs |

The GPS module is used for positioning, while the HMC5883L provides orientation
information for navigation tasks. The MPU-6050 is included for future inertial
sensor applications.

Optocouplers are placed between the ESP32 and the ESC signal lines to reduce
the effect of electrical noise generated by the propulsion system.

## Communication system

Wireless communication is implemented using the ESP32 integrated interfaces.

### Communication features

- Wi-Fi communication with the base station
- Bluetooth support for short-range operation
- Transmission of navigation and telemetry data
- Reception of remote control commands

During experimental tests, Wi-Fi was used as the main communication channel
between the ASV and the base station.

## PCB integration

The PCB centralizes the electrical connections between sensors, actuators, and
the control system.

### Integrated connections

- ESP32 microcontroller
- GPS interface
- Compass interface
- MPU-6050 interface
- ESC control outputs
- Power input connections

The schematic defines the relationship between the navigation sensors, the
control platform, and the propulsion system used by the vessel.

## Electronics installation

All electronics should be installed inside the central module of the boat.

### Installation guidelines

1. Install the PCB inside the electronics compartment.
2. Connect the navigation sensors to their respective headers.
3. Connect the ESC signal lines to the PCB outputs.
4. Connect the battery to the power input terminals.
5. Route the motor wires toward the motor supports.
6. Verify all electrical connections before powering the system.
